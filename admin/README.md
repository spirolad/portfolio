# Partie backoffice du portfolio

Permet de gérer le contenu du portfolio, notamment les différentes sections.

Permet de modifier:
* Éducation
* Expérience
* Compétences
* Projets
* Loisirs
* Contact
* Informations globales

## Génération de CV

Le but après est de pouvoir générer un CV à partir de ces données, en utilisant un template prédéfini. Cela permettra d'avoir un CV à jour et facilement modifiable.
